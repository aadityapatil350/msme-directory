# MSMEVault.in - Claude Code Context

## Project Overview
**Goal**: Build MSMEVault.in — India's #1 directory for MSME government schemes, loans & subsidies

**Tech Stack**:
- Next.js 16.1.6 (App Router) + TypeScript + React 19
- PostgreSQL + Prisma ORM (v7.4.2)
- Tailwind CSS 4 + Radix UI (shadcn/ui)
- Deployment: Vercel
